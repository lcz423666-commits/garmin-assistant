# 跑步专项知识库

> 本文件作为系统 Prompt 的参考资料，在生成跑步类运动快报时加载。
> 所有跑步数据的解读必须基于本文件中的知识。

---

## 一、跑步核心指标

### 1.1 心率区间

佳明将心率分为5个区间（基于最大心率百分比或乳酸阈心率百分比）：

| 区间 | 最大心率% | 感受 | 训练目的 |
|------|----------|------|---------|
| Z1 热身/恢复 | 50-60% | 非常轻松，可以正常交谈 | 热身、放松、恢复跑 |
| Z2 有氧基础 | 60-70% | 舒适，能说长句子 | 有氧基础建设（应占训练量的80%） |
| Z3 有氧耐力 | 70-80% | 开始感到吃力，只能说短句 | 提升有氧效率 |
| Z4 乳酸阈 | 80-90% | 辛苦，几乎不想说话 | 提升乳酸阈、提高比赛配速 |
| Z5 无氧/VO2Max | 90-100% | 全力，无法交谈 | 提升最大摄氧量、冲刺能力 |

80/20法则：大量研究表明，最优的训练心率分布是80%的训练量在Z1-Z2（轻松有氧），20%在Z4-Z5（高强度）。大部分业余跑者的错误是Z3跑太多——既不够轻松也不够高强度，训练效率最低。

### 1.2 配速区间

基于5K比赛配速的训练配速参考：

| 训练类型 | 配速参考 | 目的 |
|---------|---------|------|
| 恢复跑 | 比5K配速慢2:00-2:30/km | 促进恢复 |
| 轻松跑 | 比5K配速慢1:30-2:00/km | 有氧基础（最重要的训练类型） |
| 马拉松配速 | 比5K配速慢0:45-1:10/km | 马拉松专项 |
| 乳酸阈配速 | 比5K配速慢0:20-0:30/km | 提升阈值能力 |
| 间歇配速 | 接近或略快于5K配速 | 提升VO2Max |
| 冲刺 | 全力 | 速度和神经肌肉 |

### 1.3 步频（踏频）

| 步频(SPM) | 含义 |
|-----------|------|
| <160 | 偏低，通常意味着步幅过大，落地冲击力大，受伤风险高 |
| 160-170 | 中等，大部分慢跑者的自然步频 |
| 170-180 | 良好，跑步经济性较好 |
| 180-190 | 理想区间，公认的最经济步频范围 |
| >190 | 偏高，通常出现在高速跑或短距离冲刺中 |

步频提升是跑步技术进步的重要标志。从165提升到175-180是一个有意义的改善，会减少关节冲击力、降低受伤风险。

注意：步频和配速有关——慢跑时步频自然低于快跑。评价步频要结合配速来看。

## 二、跑步功率

部分佳明设备支持跑步功率（通过手表内置传感器或外部功率计）。

### 2.1 跑步功率与骑行功率的区别

- 跑步功率包含了克服重力和水平推进的综合功率
- 相同体能水平下跑步功率通常是骑行FTP的70-80%
- 跑步功率目前标准化程度不如骑行功率，不同设备之间数据可比性差
- 跑步功率的主要用途是监控训练一致性和疲劳，不适合像骑行那样做精确的区间训练

### 2.2 跑步效率（功率/配速比）

单位功率下能跑多快反映了跑步经济性。这个比值越高，说明跑步技术越好。长期来看，这个指标的改善比绝对功率更有意义。

## 三、跑步训练类型与数据特征

### 3.1 恢复跑

数据特征：心率Z1, 步频可以偏低, 时长20-40分钟
分析要点：不要评价"配速太慢"——恢复跑的目的就是要慢。关注心率是否控制在Z1。

### 3.2 轻松跑/有氧跑

数据特征：心率Z2, 步频165-180, 时长30-90分钟
分析要点：这应该是训练的主体。关注心率是否稳定在Z2，有没有不自觉地跑到Z3（常见错误）。
这是最重要的训练类型——建立有氧基础、提高脂肪利用率、增强心血管系统。很多跑者觉得"太慢了没用"，但80%以上的训练量应该在这个强度。

### 3.3 节奏跑/乳酸阈跑

数据特征：心率Z4, 持续20-40分钟, 或4-6×5分钟间歇
分析要点：关注心率是否在目标区间、后半段是否有明显衰退。

### 3.4 间歇跑

数据特征：高强度段心率Z4-Z5, 恢复段心率降到Z2以下, 反复多组
分析要点：关注高强度段的配速一致性、恢复段心率回落速度。恢复段心率回落越快说明心肺能力越好。

### 3.5 长距离慢跑（LSD）

数据特征：心率Z2, 时长>90分钟, 配速稳定
分析要点：关注后半段是否有心率漂移（正常现象，但漂移超过10bpm需要关注）。这类训练最大的价值是建立耐力基础和心理韧性。

## 四、跑步特有的身体指标

### 4.1 地面接触时间（GCT）

每步脚接触地面的时间（毫秒）。

| GCT(ms) | 水平 |
|---------|------|
| >280 | 较长，通常对应较慢配速 |
| 240-280 | 中等 |
| 200-240 | 较短，跑步效率较高 |
| <200 | 精英水平 |

GCT和配速强相关——不要在慢跑时批评GCT长。

### 4.2 垂直振幅

每步身体上下弹动的幅度（厘米）。

| 垂直振幅(cm) | 水平 |
|-------------|------|
| >10 | 偏高，能量浪费多 |
| 8-10 | 中等 |
| 6-8 | 良好，跑步经济 |
| <6 | 优秀 |

垂直振幅低说明更多能量用在水平推进而不是上下弹动。但同样和配速有关。

### 4.3 垂直比率

垂直振幅/步幅的比值。这个指标排除了配速的影响，更能反映跑步经济性。

| 垂直比率(%) | 水平 |
|------------|------|
| >10 | 需要改善 |
| 8-10 | 中等 |
| 6-8 | 良好 |
| <6 | 优秀 |

## 五、跑步与心率的关系

### 5.1 心率漂移在跑步中的表现

跑步中的心率漂移比骑行更明显：
- 30分钟以内：漂移3-5bpm正常
- 30-60分钟：漂移5-10bpm正常
- 60分钟以上：漂移10-15bpm可以接受

漂移幅度受温度影响很大——每升高5°C环境温度，心率漂移可能增加30-50%。

### 5.2 心率恢复速度

运动结束后1分钟内心率下降的幅度是心血管健康的重要指标：
- 下降>20bpm：良好的心血管健康
- 下降12-20bpm：正常
- 下降<12bpm：心血管适应性偏低或过度疲劳

长期来看，心率恢复速度的改善是训练效果的有力证据。

## 六、乳酸阈值

### 6.1 基本概念

乳酸阈值是运动强度增加到一定程度后，血液中乳酸开始快速积累的转折点。在这个强度以下可以长时间持续，在这个强度以上会快速疲劳。

佳明通过心率和配速的关系来估算乳酸阈值，给出：
- 乳酸阈心率（bpm）
- 乳酸阈配速（min/km）

### 6.2 乳酸阈的训练意义

- 乳酸阈是决定中长距离表现的最关键因素之一
- 半马的比赛配速通常接近乳酸阈配速
- 马拉松的比赛配速通常是乳酸阈配速的85-90%
- 提升乳酸阈是提升中长距离成绩的最直接途径

### 6.3 乳酸阈的进步参考

- 新手开始训练后，乳酸阈配速可能在3-6个月内提升30-60秒/km
- 有基础的跑者每年提升10-20秒/km是合理的进步速度
- 高水平跑者提升5-10秒/km就已经很显著

## 七、比赛预测

佳明的比赛预测基于VO2 Max和训练数据给出5K、10K、半马、全马的预测成绩。

### 7.1 使用注意

- 预测成绩假设用户以最优策略完成比赛（均匀配速、充分准备）
- 实际成绩通常比预测慢3-10%（天气、赛道、补给策略都会影响）
- 全马预测的准确度最低（影响因素最多）
- 预测值的趋势变化比绝对值更有参考价值

### 7.2 各距离之间的关系

经验法则：
- 10K成绩 ≈ 5K成绩 × 2.1-2.15
- 半马成绩 ≈ 10K成绩 × 2.2-2.25
- 全马成绩 ≈ 半马成绩 × 2.1-2.2（训练充分时）到 × 2.3-2.5（训练不充分时）

## 八、跑步伤病风险信号

### 8.1 数据中的预警信号

- 某侧地面接触时间突然增加：可能是该侧腿部有不适，身体在代偿
- 步频突然下降且配速不变：可能是步幅被迫加大，关节压力增加
- 连续多次训练后静息心率持续升高：过度训练风险
- 相同配速下心率持续升高：疲劳累积或生病前兆

### 8.2 不应该给出的建议

- 不要诊断具体伤病（"你可能有IT束综合征"）
- 不要给出具体的康复动作建议（"做这个拉伸"）
- 只能指出数据异常并建议关注身体感受

## 九、跑步恢复时间表

| 训练类型 | 恢复需要 |
|---------|---------|
| 恢复跑/轻松跑 | 12-24小时 |
| 节奏跑 | 24-36小时 |
| 间歇跑 | 36-48小时 |
| 长距离跑(>90分钟) | 48-72小时 |
| 半马比赛 | 7-14天完全恢复 |
| 全马比赛 | 14-30天完全恢复 |

恢复跑（Z1，20-30分钟）通常比完全休息更有助于恢复。

## 十、跑步与骑行的交叉训练

对于同时做骑行和跑步的用户：
- 骑行可以作为跑步的恢复手段（无冲击负荷）
- 骑行建立的有氧基础可以转移到跑步
- 但两者的肌肉募集模式不同，骑行训练不能完全替代跑步训练
- 安排交叉训练时，注意总体训练负荷不要叠加过度——骑行的TSS和跑步的训练压力要一起看

## 十一、不同天气条件下的跑步数据调整

### 11.1 高温天气（>30°C）

- 心率可能比凉爽天气高10-15bpm
- 配速可能慢15-30秒/km
- 不应该用高温天的数据来判断体能下降
- 佳明的VO2 Max算法会做热适应校正

### 11.2 寒冷天气（<5°C）

- 前5-10分钟心率可能偏高（身体还没适应）
- 适应后心率可能比常温略低
- 呼吸道冷空气刺激可能影响跑步感受

### 11.3 高海拔（>1500m）

- 心率通常升高5-10%
- 配速下降是正常的
- VO2 Max估算会受影响（佳明有海拔校正）
- 适应期通常需要2-3周
